# RealismOverhaul-7DTD
  Core mod that transforms the game into a deeper, more immersive real world simulation. Expands gameplay systems, introduces new mechanics, and unlocks advanced possibilities for other mods. Seasons, Calendars, a lot of CVars...I spent a lot of time coding a complex calendar system just so I could be precisely sure what day I’m going to die.
